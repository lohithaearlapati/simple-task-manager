# Simple Task Manager

## Project Description
This is a simple full-stack web application where users can register, log in, and manage tasks.  
The frontend and backend are connected using REST APIs, and data is stored in a MySQL database.

---

## Features
- User registration
- User login
- Add tasks
- View tasks

---

## Technologies Used
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Database: MySQL

---

## API Endpoints
- POST /register
- POST /login
- POST /task
- GET /tasks/:userId

---

## Database Tables
**users**
- id
- username
- password

**tasks**
- id
- task
- user_id
